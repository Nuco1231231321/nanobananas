Perfect Nano Banana Prompts

Learn how to write effective prompts to create amazing nano banana figures and models.

Prompt Structure Guide

A good nano banana prompt includes these elements:

1. Subject

Clearly state what you want: "nano banana figure", "nano banana model", etc.

2. Style

Specify art style: "anime", "realistic", "bandai style", "cartoon", etc.

3. Details

Add specific details: "with hat", "mechanical arms", "glowing elements", etc.

4. Quality

Include quality descriptors: "highly detailed", "8k", "photorealistic", etc.
